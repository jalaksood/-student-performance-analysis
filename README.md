# 📊 Student Performance Analysis - Mathematics Grades

## Project Description
This project analyzes student math performance data to answer key questions about grades, study habits, and gender differences.

## Analysis Questions Answered
1. **What is the average final grade (G3)?**
2. **How many students scored above 15/20?**
3. **Is study time correlated with performance?**
4. **Which gender performs better on average?**

## Files in This Repository
| File | Description |
|------|-------------|
| `student_analysis.ipynb` | Jupyter notebook with complete analysis |
| `student-mat.csv` | Dataset containing student information |
